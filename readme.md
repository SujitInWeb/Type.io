## What is Type.io?

A simple typing test application where you can practice typing and measure your speed.

**Two Modes:**
- **Infinite Mode** - Practice without time limits
- **Calculator Mode** - 30-second timed test with WPM calculation

---

## Getting Started

**Requirements:**
- Node.js (v20+)
- npm

**Installation:**

```bash
# Clone the repo
git clone https://github.com/SujitInWeb/Type.io.git

# Navigate to folder
cd Type.io

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open `http://localhost:5173` in your browser.

---

## How to Use

1. Choose a mode from the sidebar
2. Start typing the text shown
3. Watch your typo count
4. Click restart for new text
5. Toggle dark/light theme anytime
